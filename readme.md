该项目的目的是建立一个简易的生产物料管理系统，将包含以下内容：
1. 生产物料信息
   * 基础信息「包括本地编号、厂家编号、描述信息等」
   * 列表
   * 分类
   * 供应商
2. 供应商信息
   * 基础信息
